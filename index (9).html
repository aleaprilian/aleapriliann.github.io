Format Media Digital, Font, dan Film Animasi Populer

Format Gambar Populer

JPG/JPEG

Format gambar dengan kompresi lossy, ideal untuk fotografi

Ekstensi: .jpg, .jpeg Kompresi: Lossy

PNG

Format gambar dengan transparansi, ideal untuk gambar digital

Ekstensi: .png Kompresi: Lossless

GIF

Format gambar animasi dengan dukungan 256 warna

Ekstensi: .gif Mendukung animasi

BMP

Format gambar bitmap tanpa kompresi dari Microsoft

Ekstensi: .bmp Kompresi: Tidak ada

TIFF

Format gambar untuk pencetakan dan penyimpanan arsip

Ekstensi: .tiff, .tif Kompresi: Fleksibel

WebP

Format gambar modern dengan kompresi superior untuk web

Ekstensi: .webp Kompresi: Lossy/Lossless

SVG

Format vektor berdasarkan XML, skalabel tanpa kehilangan kualitas

Ekstensi: .svg Berbasis vektor

HEIC

Format gambar baru dari Apple dengan kompresi lebih efisien

Ekstensi: .heic Kompresi: Tinggi

RAW

Format gambar tanpa proses untuk fotografi profesional

Ekstensi: .raw, .cr2, .nef Tidak terkompresi

PDF

Format dokumen yang mendukung gambar dan teks

Ekstensi: .pdf Multipage

Format Video Populer

MP4 (MPEG-4)

Format paling umum dengan dukungan luas di semua perangkat

Ekstensi: .mp4
Codec Umum: H.264, H.265
Dukungan Web: Sangat Baik
Kompatibilitas Luas Streaming Kualitas Baik

AVI

Format klasik Microsoft, transcoded ke MP4 untuk playback

Ekstensi: .avi
Codec Umum: DivX, XviD
Dukungan Web: Terbatas
Kompatibilitas Lama Container Fleksibel

MKV

Format open source, transcoded ke MP4 untuk playback

Ekstensi: .mkv
Codec Umum: H.264, H.265, AV1
Dukungan Web: Terbatas
Multi-Track Subtitle Internal Fitur Lanjutan

MOV

Format asli Apple, mendukung H.264 untuk web

Ekstensi: .mov
Codec Umum: Apple ProRes, H.264
Dukungan Web: Sedang
Editing Video Kualitas Tinggi

WMV

Format Microsoft, transcoded ke MP4 untuk playback

Ekstensi: .wmv
Codec Umum: VC-1, WMV
Dukungan Web: Rendah
DRM Support Windows Friendly

FLV

Format Flash, transcoded ke MP4 untuk playback

Ekstensi: .flv
Codec Umum: VP6, H.264
Dukungan Web: Menurun
Legacy Web Streaming Era Flash

WebM

Format open source untuk web dengan kualitas tinggi

Ekstensi: .webm
Codec Umum: VP8, VP9, AV1
Dukungan Web: Sangat Baik
Open Source HTML5 Efisiensi Tinggi

AVCHD

Format kamera HD, transcoded ke MP4 untuk playback

Ekstensi: .mts, .m2ts
Codec Umum: H.264
Dukungan Web: Rendah
HD Camera Blu-ray

MPEG-2

Format DVD, transcoded ke MP4 untuk playback

Ekstensi: .mpg, .mpeg, .ts
Codec Umum: MPEG-2
Dukungan Web: Rendah
DVD Standard Broadcast TV

3GP

Format video untuk perangkat mobile lama

Ekstensi: .3gp
Codec Umum: H.263, MPEG-4
Dukungan Web: Sedang
Mobile Legacy Ukuran Kecil
